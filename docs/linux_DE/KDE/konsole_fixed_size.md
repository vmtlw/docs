# Настройка фиксированного размера Konsole в KDE

Эта инструкция поможет настроить фиксированный размер окна терминала Konsole в KDE Plasma, чтобы терминал не запоминал последний размер окна и всегда открывался с заданными размерами.

## Проблема

По умолчанию Konsole запоминает последний размер и положение окна, что может быть неудобно, если вы предпочитаете всегда иметь терминал определённого размера.

## Решение

Необходимо создать правило окна в KDE Plasma, которое будет принудительно устанавливать фиксированный размер для Konsole.

## Вариант 1: Через графический интерфейс

1. Откройте Konsole и установите желаемый размер окна
2. Нажмите Alt+F3 или кликните правой кнопкой мыши на заголовке окна
3. Выберите "Дополнительные действия" → "Особые настройки окна"
4. Перейдите на вкладку "Размер и положение"
5. Отметьте "Размер" и выберите "Принудительно" из выпадающего списка
6. Нажмите "Применить" и "OK"

## Вариант 2: Через файл конфигурации

Отредактируйте файл `~/.config/kwinrulesrc`, добавив в него следующие правила:

```ini
[7207cd3c-6935-49eb-9546-83c1322379fe]
Description=Параметры приложения для org.kde.konsole
size=800,600
sizerule=3
wmclass=konsole org.kde.konsole
wmclasscomplete=true
wmclassmatch=1

[eff619d9-7d3f-47c8-89d7-e9f40a333f3f]
Description=Параметры окна для org.kde.konsole
ignoregeometry=true
ignoregeometryrule=2
placement=4
placementrule=2
size=800,300
sizerule=2
title=~ : bash — Konsole
wmclass=org.kde.konsole
wmclasscomplete=true
wmclassmatch=1

[General]
count=2
rules=7207cd3c-6935-49eb-9546-83c1322379fe,eff619d9-7d3f-47c8-89d7-e9f40a333f3f
```

После изменения файла перезапустите KWin с помощью команды:

```bash
kwin_x11 --replace &
```

## Пояснение параметров

- `size=800,300` - задает размер окна (ширина x высота в пикселях)
- `sizerule=2` - правило для размера (2 = принудительно)
- `ignoregeometry=true` - игнорировать сохраненную геометрию окна
- `placement=4` - правило размещения окна (4 = центр экрана)

## Дополнительные настройки

Для более точной настройки можно изменить следующие параметры:

1. Размер окна: измените значение `size=800,300` на желаемые размеры
2. Положение окна: добавьте `position=x,y` и `positionrule=3` для фиксации положения
3. Отключение запоминания размера: используйте `ignoregeometry=true` и `ignoregeometryrule=2`

## Применение к другим приложениям

Аналогичным образом можно настроить фиксированный размер для любых других приложений в KDE Plasma, заменив значение `wmclass=konsole org.kde.konsole` на класс нужного приложения.
