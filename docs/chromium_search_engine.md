### Сделать google.com поиском по умолчанию в debian chromium не просто, но вот что вам поможет:

1. перейдите по URL `chrome://settings/searchEngines`
2. Рядом с пунктом «Поиск по сайту» нажмите кнопку «Добавить» ( а не рядом с пунктом «Поисковые системы», где, как вы могли бы подумать, она должна быть).
3. В появившемся диалоговом окне: в поле «Поисковая система» введите «Google», в поле «Ярлык» введите «google», в поле «URL с %s вместо запроса» введите `https://www.google.com/search?q=%s`. Обратите внимание , если на конце не будет `%s` кнопка `сделать по умолчанию` так и останется не активной
4. Нажмите кнопку «Добавить» в диалоговом окне. Это добавит Google в качестве варианта для «Поиска по сайту» (не там, где вы хотели!), поэтому
5. Нажмите на значок меню (гамбургер) в появившейся строке Google и выберите «Сделать поиском по умолчанию». Это переместит Google из раздела «Поиск по сайту» в раздел «Поисковые системы» и установит Google в качестве вашей новой поисковой системы по умолчанию. (Если вы просто хотели использовать Google в качестве варианта поисковой системы, но не по умолчанию, вам нужно будет вручную изменить поисковую систему по умолчанию на ту, которую вы хотели).
