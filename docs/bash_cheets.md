## Bash

### Сравнение целых чисел

| Оператор | Альтернатива | Описание |
|--------|-------------|----------|
| `-eq` | `=` `(( = ))` | равно |
| `-ne` | `!=` `(( != ))` | не равно |
| `-gt` | `>` `(( > ))` | больше |
| `-ge` | `>=` `(( >= ))` | больше или равно |
| `-lt` | `<` `(( < ))` | меньше |
| `-le` | `<=` `(( <= ))` | меньше или равно |

> ⚠️ Если использовать **нечисловые значения** в `[[ ]]`, сравнение происходит по **ASCII-кодам**.

### Проверка строк

- `[[ -z STRING ]]` — строка пустая  
- `[[ -n STRING ]]` — строка не пустая  

### Логические операторы

- `[[ command -a command ]]` — логическое **И** (аналог `&&`)
- `[[ command -o command ]]` — логическое **ИЛИ** (аналог `||`)

### Проверка файлов

| Оператор | Описание |
|--------|----------|
| `-e file` | файл существует |
| `-f file` | обычный файл |
| `-d file` | каталог |
| `-r file` | доступен для чтения |
| `-w file` | доступен для записи |
| `-x file` | доступен для выполнения |
| `-s file` | размер больше нуля |
| `-O file` | владелец — текущий пользователь |
| `-G file` | владелец — текущая группа |
| `-h file`, `-L file` | символическая ссылка |
| `-b file` | блочное устройство |
| `-c file` | символьное устройство |
| `-S file` | socket |
| `-N file` | изменён с момента последнего чтения |
| `file1 -ef file2` | один inode |
| `file1 -nt file2` | file1 новее |
| `file1 -ot file2` | file1 старше |

### Арифметические и логические операторы

| Оператор | Назначение |
|--------|------------|
| `id++`, `id--` | постинкремент / постдекремент |
| `++id`, `--id` | преинкремент / предекремент |
| `-`, `+` | унарный минус / плюс |
| `!`, `~` | логическое / побитовое отрицание |
| `**` | возведение в степень |
| `* / %` | умножение / деление / остаток |
| `+ -` | сложение / вычитание |
| `<< >>` | побитовые сдвиги |
| `== !=` | сравнение |
| `&` | побитовое И |
| `^` | побитовое XOR |
| `|` | побитовое ИЛИ |
| `&&` | логическое И |
| `||` | логическое ИЛИ |
| `expr ? a : b` | тернарный оператор |
| `= *= /= += -=` | присваивание |
| `,` | оператор запятая |
| `<<<` | here-string (строка как stdin) |

### Специальные переменные Bash

| Переменная | Описание |
|-----------|----------|
| `$0` | путь к исполняемому скрипту |
| `$#` | количество аргументов |
| `$@` | массив аргументов (`$1 $2 ...`) |
| `$*` | аргументы без разделителей |
| `$$` | PID текущего процесса |
| `$!` | PID последнего фонового процесса |
| `$?` | код выхода последней команды |
| `$-` | активные флаги shell |

## VIM

### Общие команды

- `/nohl` — отключить подсветку поиска
- `:e file` — открыть файл
- `:w` — сохранить
- `:q` / `:q!` — выйти / выйти без сохранения
- `:sav file` — сохранить как

---

### Поиск и замена

| Команда | Описание |
|-------|----------|
| `/text` | поиск вниз |
| `?text` | поиск вверх |
| `*` | поиск слова под курсором |
| `/\ctext` | без учёта регистра |
| `/abc\|xyz` | abc или xyz |
| `:%s/x/y/g` | заменить все x на y |
| `:%s/x/y/gc` | с подтверждением |
| `:%s/^/Begin/` | вставить в начало строки |
| `:%s/$/End/` | вставить в конец строки |
| `:%s/ *$//g` | удалить пробелы в конце |
| `:g/text/d` | удалить строки с text |
| `:v/text/d` | удалить строки без text |

---

### Навигация

| Команда | Действие |
|-------|----------|
| `h j k l` | перемещение |
| `e`, `b` | конец / начало слова |
| `gg`, `G` | начало / конец файла |
| `:80` | перейти к строке 80 |
| `%` | к парной скобке |
| `[[` | начало функции |

---

### Буферы, окна, вкладки

- `:tabnew` — новая вкладка  
- `gt` — следующая вкладка  
- `:split`, `:vsplit` — разделить окно  
- `ctrl-w w` — следующее окно  
- `ctrl-w =` — выровнять окна  

---

### Выравнивание и отступы

- `>>`, `<<` — добавить / убрать отступ
- `=%` — отформатировать блок
- `gg=G` — форматировать весь файл
- `:set autoindent`
- `:set smartindent`

---

### Автодополнение

- `Ctrl+n`, `Ctrl+p` — дополнение слова
- `Ctrl+x Ctrl+l` — дополнение строки
- `Ctrl+x Ctrl+k` — по словарю

---

### UNIX-команды внутри Vim

- `:!cmd` — выполнить команду
- `!!cmd` — вставить вывод
- `:sh` — перейти в shell
- `exit` — вернуться в Vim
