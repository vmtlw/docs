# Управление дисками

## Отключение диска

Для безопасного отключения диска в Linux существует несколько способов:

### Использование udisksctl

Современный способ отключения диска через удобную утилиту:

```bash
udisksctl power-off -b /dev/sdb
```

Где `/dev/sdb` - путь к устройству, которое вы хотите отключить.

### Использование hdparm

Альтернативный способ с помощью утилиты hdparm:

```bash
hdparm -s 0 /dev/sda
```

!!! warning "Важно"
    Перед отключением диска убедитесь, что все файловые системы на нем размонтированы, и никакие процессы не используют диск. Иначе это может привести к потере данных.

## Другие операции с дисками

### Проверка состояния диска

Для проверки состояния SMART диска:

```bash
smartctl -a /dev/sda
```

### Мониторинг производительности диска

Для мониторинга производительности дисков в реальном времени:

```bash
iostat -xz 1
```
