# Настройка поисковой системы в Chromium

## Проблема

В Debian и некоторых других дистрибутивах Linux версия Chromium не содержит Google как поисковую систему по умолчанию. Вместо этого может быть установлен Bing или DuckDuckGo. Добавление Google (или другой предпочитаемой поисковой системы) не является интуитивно понятным.

## Решение: Как добавить Google в качестве поисковой системы по умолчанию

![Chromium Search Settings](../assets/chromium-search.png)

### Шаги по настройке:

1. Откройте Chromium и перейдите по URL: `chrome://settings/searchEngines`

2. **Важно!** Нажмите кнопку «Добавить» рядом с разделом **«Поиск по сайту»** (а не рядом с разделом «Поисковые системы», что было бы логичнее)

3. В появившемся диалоговом окне заполните следующие поля:
   - **Поисковая система**: `Google`
   - **Ярлык**: `google`
   - **URL с %s вместо запроса**: `https://www.google.com/search?q=%s`
   
   > **Обратите внимание!** Символ `%s` в конце URL обязателен, иначе кнопка «Сделать по умолчанию» останется неактивной

4. Нажмите кнопку «Добавить» в диалоговом окне

5. В появившейся строке Google нажмите на значок меню (три точки) справа и выберите «Сделать поиском по умолчанию»

После этих действий Google переместится из раздела «Поиск по сайту» в раздел «Поисковые системы» и будет установлен в качестве системы поиска по умолчанию.

## Добавление других поисковых систем

По этому же принципу можно добавить любую другую поисковую систему. Некоторые популярные варианты:

### DuckDuckGo
```
https://duckduckgo.com/?q=%s
```

### Yandex
```
https://yandex.ru/search/?text=%s
```

### Bing
```
https://www.bing.com/search?q=%s
```

## Дополнительные настройки

После добавления поисковой системы вы можете:

1. Настроить ярлыки для быстрого доступа (например, набрав "y" и нажав Tab для поиска через Yandex)
2. Изменить порядок поисковых систем, перетаскивая их в списке
3. Удалить ненужные поисковые системы через меню с тремя точками
